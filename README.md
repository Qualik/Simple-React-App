# A Simple React App

Link to App - https://qualik.github.io/Simple-React-App

A React 'Name & Job' title App that allows you to:

- Add name
- Add job
- Submit name
- Submit job
- Delete name
- Delete job

This shows the fundamentals of React:

- Simple / Custom Class Componenets
- State
- Props
- Submitting form data
- JSX
